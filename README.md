# tdlog-boardgame-ai

## Requirements
- `python3-dev`
- `cmake`

## Installation

``` bash
# Compile pybind11 first
cd pybind11
mkdir build
cd build
cmake ..
make -j 4

# Get back to work
cd ../..
```

Make sure that `boardgame_ai_py.so` is available in the root directory or installed as a python package.
